#  Documentação Técnica — FlexCorp Ecommerce

Este repositório contém a **documentação completa** do sistema de e-commerce desenvolvido para a loja **“Um Convite de Casamento”**. Aqui você encontrará informações sobre os requisitos do sistema, modelagem, diagramas, estrutura do banco de dados, fluxos de navegação e outros documentos técnicos.

---

##  Observações

- A documentação acompanha o desenvolvimento do sistema — novas versões podem ser adicionadas conforme o projeto evoluir.
- Alguns tópicos ainda estão em revisão e podem receber atualizações com feedback da equipe e orientações acadêmicas.
